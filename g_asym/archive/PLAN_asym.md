**Objective:** Produce a LaTeX table showing how demand multipliers differ across shock-size bins for positive vs. negative demand shocks, mirroring the "Panel B: Coefficient differences" format of the main price-impact table.

**Methodology:** Three parallel workflows:

1. **v1 (original, differences only):** Read pre-computed FM results from `fm_stdev_pos.RDS` / `fm_stdev_neg.RDS`. Extract pairwise bin-coefficient differences for spec_idx 1–3, types BMI/FIT/OFI. Format as a two-panel LaTeX table with significance stars. Script: `table_asym_stdev.py`.

2. **v2 (pooled σ, raw multipliers + differences):** Re-run FM regressions on sign-specific subsamples (dropping, not zeroing, opposite-sign rows) using pooled σ bin thresholds from `reg_table_static.RDS`. Generates both raw multipliers and differences per panel. Script: `code/g_asym/2a_regression_fm_asym_v2.R`. Output: `code/g_asym/reg_asym_stdev_v2.tex`.

3. **v3 (sign-specific σ robustness):** Re-run the v2 row-dropping specification, but recompute σ bins within each sign-specific subsample by `yyyymm` and `type` before estimating the stdev regressions. Generates raw multipliers and differences per panel. Script: `code/g_asym/2a_regression_fm_asym_v3.R`. Output: `code/g_asym/reg_asym_stdev_v3.tex`.

### Data Quality Notes
- v1 files generated by `code/R/_archive_code/2a_regression_fm_asym_AC.R`. The original script zeros out opposite-sign OFI (keeps rows as zero-demand observations via `fifelse(ofi>0, ofi, 0)`) and computes σ on the zeroed distribution. This produces a much smaller σ than the pooled spec, causing inflated multipliers (~2–3× larger than pooled results).

- Types present: `BMI`, `FIT`, `OFI`. `spec_idx` 1/2/3 = no controls / char controls / char+liq controls.

- [x] **Step 4: Analyze — format cells and build LaTeX table**

```python
# %% [markdown]
"""
## Format cells with stars, build two-panel LaTeX tabular.
Star thresholds match reg_static_fm_stdev.Rmd: [2.576, 1.960, 1.645] for 1/5/10%.
"""

# %%
THRESHOLDS = [2.576, 1.960, 1.645]

def format_coef(coef, se):
    abs_t = abs(coef / se)
    n_stars = sum(abs_t > t for t in THRESHOLDS)
    stars = '^{' + '*' * n_stars + '}$' if n_stars > 0 else '$'
    return f'${coef:.2f}{stars}'

def format_se(se):
    return f'$({se:.2f})$'

ROW_LABELS = {
    'ofi_bin2 - ofi_bin1': r'$M_{\{|d_{n,t}| \in [\sigma, 2\sigma]\}} - M_{\{|d_{n,t}| < \sigma\}}$',
    'ofi_bin3 - ofi_bin2': r'$M_{\{|d_{n,t}| > 2 \sigma\}} - M_{\{|d_{n,t}| \in [\sigma, 2\sigma]\}}$',
    'ofi_bin3 - ofi_bin1': r'$M_{\{|d_{n,t}| > 2 \sigma\}} - M_{\{|d_{n,t}| < \sigma\}}$',
}
TYPES = ['BMI', 'FIT', 'OFI']

def build_panel_rows(df):
    """Return list of (label, [9 coef cells], [9 se cells]) tuples."""
    rows = []
    for var in DIFF_VARS:
        coef_cells, se_cells = [], []
        for type_name in TYPES:
            for spec in SPEC_IDX:
                row = df[(df['type'] == type_name) & (df['spec_idx'] == spec) & (df['var'] == var)]
                c, s = row['coef'].values[0], row['se'].values[0]
                coef_cells.append(format_coef(c, s))
                se_cells.append(format_se(s))
        rows.append((ROW_LABELS[var], coef_cells, se_cells))
    return rows

def fmt_row(label, cells, add_vspace=False):
    vspace = r' \vspace{5pt}' if add_vspace else ''
    return f'  {label} & {" & ".join(cells)} \\\\{vspace} \n'

def build_tabular(pos_rows, neg_rows):
    col_header = '  & (1) & (2) & (3) & (4) & (5) & (6) & (7) & (8) & (9) \\\\ \n'
    group_header = (
        r'  \cmidrule(l){2-4} \cmidrule(l){5-7} \cmidrule(l){8-10}' + ' \n'
    )
    measure_header = (
        r'  & \multicolumn{3}{c}{BMI} & \multicolumn{3}{c}{FIT} & \multicolumn{3}{c}{OFI} \\' + ' \n'
    )
    dep_var_header = (
        r'  \cmidrule(l){2-10}' + ' \n'
    )

    lines = [r'\begin{tabular}{lccccccccc}' + '\n']

    for panel_label, rows in [('Panel A: Positive shocks', pos_rows),
                               ('Panel B: Negative shocks', neg_rows)]:
        lines.append(f'  \\hline \\multicolumn{{10}}{{c}}{{{panel_label}}} \\\\\n')
        lines.append(' \\hline\n')
        lines.append(measure_header)
        lines.append(group_header)
        lines.append(col_header)
        for i, (label, coef_cells, se_cells) in enumerate(rows):
            is_last = (i == len(rows) - 1)
            lines.append(fmt_row(label, coef_cells, add_vspace=is_last))
            lines.append(fmt_row('', se_cells))

    lines.append('  \\hline \n')
    lines.append(r'\end{tabular}' + '\n')
    return ''.join(lines)

pos_rows = build_panel_rows(pos_filt)
neg_rows = build_panel_rows(neg_filt)
tabular = build_tabular(pos_rows, neg_rows)
print(tabular[:500])  # spot-check first 500 chars
```

**Key change from v2:** After filtering to the sign-specific subsample, override the pooled bin columns with bins computed from the subsample's own σ:

```r
# Insert after data_all <- data_all[ofi > 0] / [ofi < 0]
data_all[, sd_ofi_sub := sd(ofi), .(yyyymm, type)]
data_all[, bin := 1]
data_all[abs(ofi) > sd_ofi_sub,     bin := 2]
data_all[abs(ofi) > 2 * sd_ofi_sub, bin := 3]
data_all[, ofi_bin1 := ifelse(bin == 1, ofi, 0)]
data_all[, ofi_bin2 := ifelse(bin == 2, ofi, 0)]
data_all[, ofi_bin3 := ifelse(bin == 3, ofi, 0)]
data_all[, sd_ofi_sub := NULL]
```

**Tradeoff (documented):** σ_pos ≠ σ_neg in general, so bin thresholds differ across panels. Multipliers within each panel answer "does impact decline relative to typical shocks of that sign?" but are not directly comparable across panels. v2 (pooled σ) remains the cleaner main-paper spec; v3 is a robustness check. Placement (main paper vs appendix) deferred until results are seen.
